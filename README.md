# Tkinter メモアプリ

Tkinterで作成した画面付きのメモアプリです。

## できること
- メモの追加
- メモの削除
- メモの編集
- メモの更新
- 完了 / 未完了の切り替え
- 検索
- 一覧に戻す
- 日時付き表示
- 保存 / 読み込み
- スクロールバー対応

## 使用技術
- Python
- Tkinter

## 実行方法
- python tk_memo_app.py

## 保存ファイル
- memo.txt

## 工夫した点
- 画面付きで操作しやすくした
- 検索機能を追加した
- 完了 / 未完了の切り替えを追加した
- スクロールバーを付けて見やすくした
- 成功メッセージは自動で消える表示にして使いやすくした
